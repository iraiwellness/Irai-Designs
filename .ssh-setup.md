# SSH setup for Irai-Designs (GitHub)

Use this when pushing to `git@github.com:iraiwellness/Irai-Designs.git`.

## Option A — Use your existing key (simplest)

You already have `~/.ssh/id_ed25519`. Add the **public** key to GitHub:

```bash
cat ~/.ssh/id_ed25519.pub
```

1. Copy the output.
2. GitHub → **Settings → SSH and GPG keys → New SSH key**
3. Paste the key and save.
4. Ensure your GitHub user is a member of the **iraiwellness** org with write access to `Irai-Designs`.

Test:

```bash
ssh -T git@github.com
# Hi <username>! You've successfully authenticated...
```

## Option B — Dedicated key for Irai (recommended if you use multiple GitHub accounts)

```bash
# 1. Generate a key for Irai (use your personal email tied to iraiwellness access)
ssh-keygen -t ed25519 -C "your-personal-email@example.com" -f ~/.ssh/id_ed25519_irai

# 2. Start agent and add key
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519_irai

# 3. Copy public key → add to GitHub (Settings → SSH keys)
cat ~/.ssh/id_ed25519_irai.pub
```

Create `~/.ssh/config`:

```
Host github.com-irai
  HostName github.com
  User git
  IdentityFile ~/.ssh/id_ed25519_irai
  IdentitiesOnly yes
```

Then set the remote:

```bash
git remote set-url origin git@github.com-irai:iraiwellness/Irai-Designs.git
```

Test:

```bash
ssh -T git@github.com-irai
```

## Push after SSH works

```bash
cd /path/to/Irai-Design
git push -u origin main
```
